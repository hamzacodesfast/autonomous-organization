# Autonomous Organization print assets

Local No. 001 uses the primary mark on the back only.

Use this file for the Printify draft product:

- `local-001-back-mark-offwhite.png`

Do not upload:

- social banners
- social avatar/logo exports
- product mockups
- embroidery templates

The SVG source is editable but should not be uploaded directly to Printify because Printify's SVG support does not preserve live text layers. Upload the rendered PNG unless a human converts the SVG text to paths in a vector editor first.

Run `npm run export:print` from the repository root after editing the source SVG.
